# CANOPY: First Night / 雨林第一夜

> 类型：原创浏览器硬核生存纵切片  
> 单局时长：24–28 分钟  
> 平台：桌面浏览器，键盘与鼠标优先  
> 目标：用最小内容量证明“感知—诊断—准备—风险—恢复—知识增长”能形成完整、好玩的生存体验。

## 1. 一句话提案

玩家是一名被暴雨困在雨林深处的野外调查员。第一天下午必须建立能挡雨、点火和休息的营地；第二天根据纸图与坐标前往废弃气象站取回电池，在蛇和大型捕食者的威胁下带伤返营，启动求救信标。

本作不讲 Green Hell 的角色或故事，也不复制其地图、界面绘图、资产或文本。它是一款原创机制纵切片。

## 2. 玩家承诺

一局结束后，玩家应能说出：

- “我开始能听懂这片雨林了。”
- “那个问题不是随机扣血，是我更早之前做错了选择。”
- “我知道下次出发前应该带什么。”
- “回到火边的那一刻真的松了一口气。”

如果玩家只说“我把几个条补满了”或“我按提示依次点完了”，设计未达到目标。

## 3. 设计支柱

### 3.1 知识比装备重要

新配方、食物效果、声音含义和治疗方法，是主要成长内容。工具只扩大玩家运用知识的能力。

### 3.2 一个选择至少影响两个系统

脏水同时解决即时脱水并制造寄生虫风险；火同时解决净水、烹饪、光和理智，又依赖燃料与遮雨；超重不仅降低速度，还会令返程时更难逃跑。

### 3.3 危险必须可感知、可诊断、可反制

死亡可以严厉，但不能无提示。声音、植被扰动、伤口外观和状态变化必须给玩家理解因果的机会。

### 3.4 营地是一台安全感机器

营地不是装饰。它应同时改变火、雨水、睡眠、烹饪和物资整理规则。

### 3.5 浏览器摩擦只保留有意义的部分

保留承重、装载和准备取舍；删除像素级摆放、细小物品难点选、重复拖拽等无意义操作。

## 4. 反目标

本纵切片不追求：

- 大型开放世界。
- 写实第一人称 3D 复刻。
- 数十种食物、敌人和建筑。
- 原作角色、部落、故事或地点。
- 建造沙盒、合作、生存天数排行榜。
- 靠随机事件堆叠制造难度。
- 只有背景图和菜单按钮的资源计算器。

## 5. 核心循环

### 5.1 主循环

    观察与听声
        ↓
    手表 / 身体检查 / 纸图
        ↓
    判断最紧急问题
        ↓
    选择路线和负重
        ↓
    离营采集、试验、探索
        ↓
    遇险或受伤
        ↓
    诊断并使用匹配处理
        ↓
    回营烹饪、净水、休息
        ↓
    笔记增加，下一次决策更好

### 5.2 三层时间循环

| 时间尺度 | 玩家问题 | 示例 |
|---|---|---|
| 10–60 秒 | 我现在该停、跑、检查还是互动？ | 听到蛇响后停步；看到检查图标后查看左腿 |
| 2–6 分钟 | 当前最危险的需求怎么解决？ | 找容器接雨；做敷料；在雨前给火搭棚 |
| 10–28 分钟 | 如何扩大安全半径并完成目标？ | 建首夜营地；准备远征；取电池后返营 |

## 6. 系统因果

### 6.1 营养链

    饮食不平衡
    → 身体状态上限下降
    → 耐力恢复变慢
    → 点火、采集、逃跑更困难
    → 一次受伤更接近致命

### 6.2 脏水链

    立即喝溪水
    → 水分暂时恢复
    → 寄生虫
    → 三类食物营养下降加快
    → 必须改变原定路线寻找抗寄生资源

### 6.3 伤口链

    大型猫科命中
    → 肢体撕裂伤与出血
    → 未检查 / 脏状态下草率包扎
    → 感染
    → 发热
    → 水分和能量消耗翻倍
    → 无法安全返营

### 6.4 天气链

    暴雨
    → 雨水容器快速获得清水
    → 无遮蔽火源熄灭
    → 不能煮水、烹饪和恢复理智
    → 遮雨棚从可选建筑变成基础设施

### 6.5 理智链

    黑夜 + 蚂蟥/伤病 + 远离火
    → 理智下降
    → 错误声音和假物品
    → 玩家更难区分真实资源与风险

### 7.1 节奏保护

- 第一只大型捕食者只会在玩家获得电池后出现。
- 第一夜不会生成大型捕食者，避免建筑教学被中断。
- 第一次蛇遭遇在步行状态下至少给 1.5 秒声音预警。
- 同一时刻最多有一个重大主动威胁。
- 若玩家严重落后，暴雨可以延后最多 90 秒，但不能取消。

### 8.2 两条主路线

**安全长路：营地 → 坚果坡 → 山脊地标 → 气象站**

- 路程约比捷径长 30%。
- 可获得脂肪、长树枝和较清晰视野。
- 消耗更多水分和能量。

**危险捷径：营地 → 溪流 → 蛇草 → 气象站**

- 距离短，可在天气恶化前返回。
- 有蛇与泥泞减速。
- 路上可取得毒伤敷料原料。

玩家也可以去程走安全路、返程走捷径，或反之。

### 8.3 导航规则

- 纸图显示地形、网格和手绘地标，但没有玩家位置点。
- 手表显示当前位置网格、朝向和时间。
- 不提供常驻小地图、寻路轨迹或世界空间任务箭头。
- 关键地标在视听上必须唯一：急流声、白色岩壁、雷击枯木、气象塔金属鸣响。

## 9. 目标结构

### 即时目标：30 秒至 3 分钟

- 找水。
- 制作切割工具。
- 检查身体异常。
- 在雨前取得干纤维。

### 中期目标：5 至 14 分钟

- 建遮雨棚。
- 建叶床。
- 保住一堆火。
- 吃到至少两类营养并安全睡过第一夜。

### 全局目标：14 至 28 分钟

- 根据纸图和坐标找到气象站。
- 取回信标电池。
- 活着返回营地并发送求救。

### 胜负条件

胜利：

1. 电池安装到营地求救机。
2. 玩家当前生命大于 0。
3. 求救发送交互完成。

失败：

- 当前生命归零。
- 在水中能量归零并溺水。
- 可选挑战模式下，第二日午夜前未发出求救。

普通首局不使用硬时间失败，只通过天色和资源压力推动行动。

## 11. 生理系统

所有数值范围为 0–100。以下是首轮调参起点，不是原作数值。

### 11.1 初始状态

| 指标 | 初值 | 意图 |
|---|---:|---|
| 碳水 | 68 | 不是最紧急，但首夜前需要补充 |
| 脂肪 | 52 | 促使玩家注意坚果坡 |
| 蛋白质 | 55 | 使烹饪鱼有明确价值 |
| 水分 | 38 | 第一个紧急问题 |
| 当前生命 | 53 | 与初始身体上限接近 |
| 能量 | 72 | 足够搭营，不能无限冲刺 |
| 耐力 | 100 | 短时行动资源 |
| 理智 | 76 | 首夜忽视火和伤病时能看到轻度效果 |
| 清洁度 | 62 | 过溪和劳动后进入感染风险区 |

### 11.2 身体状态上限

身体状态上限：

    0.25 × 碳水 + 0.25 × 脂肪 + 0.25 × 蛋白质 + 0.25 × 水分

- 当前生命不能恢复超过身体状态上限。
- 若营养下降令上限低于当前生命，超出部分以每 10 秒 1 点缓慢衰减，不做瞬间截断。
- 无出血、无发热且平均营养高于 45 时，生命每分钟恢复 0.4。
- 水分归零后，额外每分钟失去 4 点生命。

### 11.3 基础消耗

| 指标 | 每现实分钟基础消耗 | 修正 |
|---|---:|---|
| 碳水 | 0.55 | 高强度劳动 ×1.35 |
| 脂肪 | 0.35 | 发热 ×1.2 |
| 蛋白质 | 0.40 | 发热 ×1.2 |
| 水分 | 1.20 | 冲刺 ×1.35；发热 ×2 |
| 能量 | 0.70 | 冲刺额外 -2.2/分钟；砍伐和点火另扣 |

寄生虫每层令三类食物营养消耗乘以 1.25，最多两层。

### 11.4 耐力

- 冲刺、砍伐、拉弓和投矛消耗耐力。
- 耐力恢复同时受身体状态上限和能量影响：

    基础恢复 ×（0.35 + 0.65 × 身体状态上限百分比）
             ×（0.45 + 0.55 × 能量百分比）

- 耐力归零只暂时禁止高强度动作；能量归零才会昏厥。

### 11.5 食物初值

| 食物 / 饮品 | 主要效果 | 风险 |
|---|---|---|
| 香蕉 | 碳水 +24 | 会腐坏，但本局只做轻量计时 |
| 林地壳果 | 脂肪 +20，蛋白 +3 | 重量较高 |
| 熟河鱼 | 蛋白 +26，能量 +6，理智 +3 | 生吃有寄生虫风险 |
| 雨水 / 煮水 | 水分 +35 / +40 | 溪水直接喝可能得寄生虫 |
| 蓝帽菌 | 移除 1 层寄生虫，能量 +4 | 首次效果未知，食用后记录 |

## 12. 身体状态与治疗

| 状态 | 来源 | 可见反馈 | 后果 | 处理 |
|---|---|---|---|---|
| 蚂蟥 | 过溪、暴雨后穿过湿叶 | 先出现检查提示；肢体上可见 | 每只持续降低理智 | 在身体检查中逐只移除 |
| 寄生虫 | 喝溪水、生鱼、低清洁度进食 | 延迟出现胃部状态图标 | 三类食物营养消耗加快 | 蓝帽菌；避免继续脏手进食 |
| 毒伤 | 蛇咬 | 肢体双孔伤、毒层数 | 生命缓降并引发发热 | 苦烟叶敷料，随后休息 |
| 撕裂伤 | 大型猫科命中 | 肢体深色裂口、流血 | 每分钟失血 3；可能感染 | 灰烬敷料最佳；普通叶绷带只暂缓 |
| 感染 | 撕裂长期未治，或低清洁度包扎 | 红肿、脓和检查提示 | 产生发热，生命恢复停止 | 蛆清创 30 秒，再用灰烬敷料 |
| 发热 | 毒伤或感染的次级状态 | 屏幕热晕、呼吸、状态环 | 水分和能量消耗 ×2，理智下降 | 处理病因后随休息消退 |

公平规则：

- 感染只在撕裂未有效处理 90 秒后判定。
- 清洁度低于 50 会增加感染概率，低于 40 时进食有寄生虫风险。
- 玩家首次得到某状态前，对应处理资源必须已在可达区域生成。
- 身体检查时世界速度降到 20%，不完全暂停。
- 错误治疗给出可理解反馈，例如“出血减慢，但伤口仍在发热”，而非静默失败。

## 13. 理智

### 13.1 下降

- 每只未移除蚂蟥：每分钟 -0.4。
- 发热：每分钟 -1.0。
- 黑夜且远离火：每分钟 -0.6。
- 睡地面：一次 -6。
- 蛆清创：一次 -3。

### 13.2 恢复

- 火边：每分钟 +1.2。
- 叶床完成睡眠：+8。
- 熟食：+2 至 +3。
- 回到主营地安全半径：只停止夜间额外下降，不直接加数值。

### 13.3 阈值

- 40 以下：耳语、错误方向的轻微枝叶声、轻微色偏。
- 20 以下：生成少量假资源轮廓；玩家接近时消散。
- 0：强烈视听干扰，但纵切片不生成复杂幻觉敌人。

无障碍模式可以关闭耳语内容，只保留非语言化失真。

## 14. 采集、背包与制作

### 14.1 背包

- 舒适承重 16 kg。
- 16–20 kg：移动速度 -20%，耐力恢复 -25%。
- 超过 20 kg：不能冲刺，但仍可丢弃物品恢复。
- 三个外置工具位：切割工具、长武器、点火工具。
- 内部使用固定网格和堆叠，不要求玩家旋转物品。
- 大型建筑材料按一束携带，不做像素级物理摆放。

### 14.2 负重取舍

远征前，玩家无法同时舒适携带：

- 两份水。
- 两份食物。
- 长矛。
- 三类医疗处理。
- 电池返程重量。

取得电池后增加 3 kg，迫使玩家丢弃补给、改走短路，或接受超重风险。

### 14.3 十一个配方

| 配方 | 材料 | 解锁 |
|---|---|---|
| 石刃 | 小石 ×2 | 开局笔记提示 |
| 石斧 | 小石 ×1 + 树枝 ×1 | 同时持有两种材料 |
| 手钻 | 树枝 ×1 + 小树枝 ×1 | 查看火页 |
| 弱矛 | 长树枝 ×1 | 首次拾取长树枝 |
| 半椰壳 | 椰子 + 切割工具 | 首次获得椰子 |
| 叶绷带 | 宽叶 ×1 | 首次检查伤口 |
| 灰烬敷料 | 叶绷带 + 灰烬 | 火烧尽后取得灰 |
| 苦烟叶敷料 | 叶绷带 + 苦烟叶 | 拾取植物或诊断毒伤 |
| 小火 | 小树枝 ×4 + 树枝 ×4 | 开局笔记提示 |
| 遮雨棚 | 长树枝 ×3 + 绳 ×2 + 棕榈叶 ×6 | 开局仅给轮廓提示 |
| 叶床 | 棕榈叶 ×6 | 首次查看睡眠提示 |

制作界面允许：

- 从背包快速加入符合配方的材料。
- 清楚展示数量不足的材料。
- 未知配方只显示可能的轮廓，不直接给完整答案。
- 已掌握配方支持一键重复制作。

## 15. 营地

### 小火

- 用于烹饪、煮水、提供光源和恢复理智。
- 点火消耗 4 点能量与一份干纤维。
- 暴露在重雨中 8 秒后熄灭。
- 位于完整遮雨棚投影内时不受雨影响。
- 燃料耗尽后留下灰烬，形成医疗配方来源。

### 遮雨棚

- 给火和叶床提供有效遮雨区。
- 棚内雨声、亮度和地面湿度明显变化。
- 建成后点亮主营地安全边界，但不阻止动物进入。

### 叶床

- 恢复能量至至少 88。
- 理智 +8。
- 推进到次日清晨。
- 睡地面只恢复至 65，理智 -6，不触发安全睡眠教学完成。

### 半椰壳

- 可放在地面接雨。
- 重雨中每 30 秒收集 20 单位清水，上限 40。
- 可放到火上煮溪水或煮鱼汤。

## 16. 时间与天气

- 开局为第一日 13:20。
- 现实一分钟约推进 40 个游戏分钟，但睡眠使用跳时。
- 8:30–10:00 现实时间之间触发暴雨，依据玩家建造进度最多延迟 90 秒。
- 暴雨开始前 60 秒依次出现天色变暗、远雷、风声和叶面翻动。
- 重雨持续 2.5–3.5 分钟。
- 第一夜在 10–12 分钟阶段进入明显黑暗。
- 使用叶床后跳到第二日 06:30。
- 返程阶段天色再次转暗，但不触发第二场完整暴雨。

天气不是纯视觉层：至少必须影响火、接雨、可见度、声音遮蔽和地面移动。

## 17. 威胁与战斗

### 17.1 蛇

- 位于蛇草捷径的若干候选点，每局只激活一个。
- 玩家步行接近时，至少提前 1.5 秒发出可定位警告声。
- 玩家继续进入攻击半径才会被咬。
- 冲刺会遮蔽部分预警并缩短反应时间。
- 可绕行、投石惊动后等待离开，或承受毒伤换取时间。

### 17.2 大型猫科

- 只在取得电池后，由返程导演选择一个合理位置生成。
- 第一次攻击前至少有 3 秒低吼和植被扰动。
- 玩家面向它、保持后退会延缓扑击；转身冲刺会加速追击。
- 投矛命中头部可迅速解决，但矛可能丢失。
- 也可借溪流、岩壁或安全长路规避。
- 首次命中造成一处撕裂，而非无条件即死；后续失血和错误治疗才构成灾难链。

### 17.3 威胁导演约束

- 不同时激活蛇和大型猫科追击。
- 玩家处于身体检查界面时，不发起新的攻击。
- 当前生命低于 20 时，捕食者不会在玩家视野外重新瞬移截路。
- 每个危险种子都必须存在至少一种无战斗通过方式。

## 18. 知识与笔记

笔记分为：

1. 当前任务。
2. 制作。
3. 植物与食物。
4. 伤病与处理。
5. 地图发现与个人记录。

记录规则：

- 拾取新材料只记录外观和地点，不自动公布全部效果。
- 食用后记录本次观察到的营养或副作用。
- 正确治疗后记录完整处理；错误治疗记录“无效/部分有效”。
- 气象站医疗记录可补足感染处理，但不直接给出所有生存答案。
- 正常模式在死亡后保留已经验证的知识，让失败成为长期进度。
- 可选“失忆挑战”关闭跨局知识保留。

## 19. UI 与交互反馈

### 19.1 控制

| 按键 | 行为 |
|---|---|
| WASD | 移动 |
| Shift | 冲刺 |
| E | 互动 / 放置材料 |
| 鼠标左键 | 使用工具 / 攻击 |
| 鼠标右键 | 瞄准 / 取消 |
| F | 手表：营养、时间、坐标/罗盘切换 |

| N | 笔记 |
| B | 身体检查 |
| Tab | 背包与制作 |
| 1–3 | 工具位 |

### 19.2 手表

- 默认不常驻四类营养数值。
- 抬腕时显示四段弧线；精确数字可在无障碍设置中开启。
- 营养下降造成的身体上限收缩要在生命条上以灰色缺口可视化。
- 吃东西时同时显示“营养上升”和“生命上限扩大”，强化因果。

### 19.3 身体检查

- 首次只显示通用检查图标，不直接说“左腿有蚂蟥”。
- 界面显示四个肢体视图；异常位置有克制的视觉标识。
- 玩家选择伤处后，再从背包拖入或快捷使用治疗物。
- 已发现知识显示匹配提示；未知处理只显示物品描述，不显示正确答案。
- 错误处理必须提供视觉或文本反馈。

### 19.4 音频

音频必须传递信息：

- 溪流方向。
- 远雷和暴雨倒计时。
- 棚内外雨声差异。
- 火将熄灭的嘶鸣。
- 蛇的方位。
- 大型猫科接近距离和方向。
- 低耐力喘息、低生命心跳和低理智失真。

所有声音必须原创或明确许可。

### 19.5 死亡因果复盘

死亡界面显示事件链，而不只显示最终伤害：

    13:42 直接喝溪水
    → 寄生虫 ×2
    → 蛋白下降加快
    → 身体状态上限降至 46
    → 返程超重，未能甩脱捕食者
    → 撕裂伤未有效处理
    → 感染与发热
    → 死亡

同时给出一个已经被玩家发现的改进方向，不泄露尚未发现的全部答案。

### 19.6 胜利结算

显示：

- 完成时间与路线。
- 最低身体状态。
- 正确诊断次数。
- 避免的战斗与发生的战斗。
- 本局学会的笔记。
- 一个未探索的替代方案，鼓励重玩。

## 20. 无障碍

- 危险音效字幕：显示类别和强弱，可选显示大致方向。
- 色觉模式：四类营养同时使用颜色、图形和文字缩写。
- 精确数值模式：允许手表显示数字。
- 减少恶心表现：降低伤口细节、呕吐和蛆动画。
- 降低心理音频：把耳语替换成非语言化失真。
- 字体缩放与高对比 UI。
- 允许重绑定按键。
- 身体检查界面默认时间减速；辅助模式可完全暂停。

## 21. 公平性与难度

### 21.1 新手保护不是取消后果

- 第一次蚂蟥事件无直接生命伤害，用于教学检查。
- 第一次蛇攻击有完整预警。
- 大型捕食者在玩家获得制作长矛机会后才出现。
- 对应治疗物永远先于状态进入可达区域。
- 没有教程弹窗时，环境、动画和短句提示仍能表达因果。

### 21.2 变化来源

每局变化：

- 蛇的候选点。
- 苦烟叶和抗寄生菌的位置。
- 暴雨开始时间 ±90 秒。
- 捕食者返程截路位置。
- 部分食物刷新点。

每局稳定：

- 治疗原理。
- 地标关系。
- 两条主路线。
- 主要目标地点。
- 所有危险的最短预警时间。

### 21.3 目标难度

- 初次玩家通关率：35%–55%。
- 第二次玩家通关率：60%–75%。
- 第一次死亡不早于 5 分钟，除非玩家主动反复进入已警告危险。
- 任何死亡都应至少能追溯到一个此前可改变的选择。

## 23. Definition of Done

### 23.1 核心功能

- [ ] 从开始到胜利或死亡可完整游玩，无开发者命令。
- [ ] 一局自然时长稳定在 24–28 分钟。
- [ ] 四类营养真实影响身体上限和耐力，而非四个独立装饰条。
- [ ] 水分归零、寄生虫、发热和出血均有可观察后果。
- [ ] 至少一条“伤口 → 感染 → 发热”因果链可在正常游玩中发生。
- [ ] 玩家必须在身体检查界面定位并处理至少一种异常。
- [ ] 火、遮雨棚、叶床和半椰壳都实际改变规则。
- [ ] 暴雨能熄灭无遮蔽火，并能填充雨水容器。
- [ ] 纸图没有玩家位置点；坐标和地标足以完成导航。
- [ ] 安全长路和危险捷径均可完成任务。
- [ ] 蛇和大型猫科均可战斗或不战斗通过。
- [ ] 笔记会因发现、试吃和治疗结果更新。
- [ ] 死亡显示可读的因果链。
- [ ] 胜利必须要求取回电池并返营发报。

### 23.2 首局体验

- [ ] 首次有意义选择不晚于 30 秒。
- [ ] 首件工具中位完成时间不超过 3 分钟。
- [ ] 首次身体检查发生在第 8 分钟前。
- [ ] 第 12 分钟前完成一次“采集—加工—消费或治疗”全循环。
- [ ] 一局至少包含五次真实取舍，不存在唯一按钮序列。
- [ ] 不阅读外部 Wiki 也能通过观察、试验和笔记通关。
- [ ] 首次测试者中至少 70% 注意到一次危险音频。
- [ ] 死亡测试者中至少 80% 能说出直接死因。
- [ ] 至少 70% 能指出一个更早、可改变的错误决定。
- [ ] 通关测试者能描述营地与离营时不同的情绪。

### 23.3 公平性

- [ ] 每个随机种子都有无战斗胜利路径。
- [ ] 每种状态的反制资源都能在致死前到达。
- [ ] 不存在无声音、无视觉先兆的致命袭击。
- [ ] 不同时激活两个重大主动威胁。
- [ ] 身体检查、笔记和地图打开时不会触发新的偷袭。
- [ ] 超重、缺药和走错路都允许通过丢物、绕行或撤退补救。

### 23.4 技术质量

- [ ] 当前 Chrome 与 Edge 桌面版完成全流程无阻断错误。
- [ ] 1280×720 及以上视口 UI 不遮挡关键交互。
- [ ] 目标硬件保持稳定帧率；逻辑固定步进与渲染帧率解耦。
- [ ] 浏览器失焦会暂停游戏，不继续消耗状态。
- [ ] 音频在首次用户手势后正确启动。
- [ ] 本地存档可恢复到最近营地或阶段节点。
- [ ] 重开不会残留上局敌人、计时器或状态。
- [ ] 控制台无未处理异常。

### 23.5 资产与发布

- [ ] 标题使用 CANOPY: First Night / 雨林第一夜。
- [ ] 不含 Green Hell 官方 Logo、截图、地图、图标、模型、音频、角色或故事文本。
- [ ] 所有第三方字体、声音和视觉资产均有明确许可记录。
- [ ] 页面和仓库注明独立、非官方机制研究，与 Creepy Jar 无隶属关系。
- [ ] 产品中的全部文案均为原创。

## 24. 纵切片完成后的扩展顺序

只有上述 DoD 通过后，才按以下顺序扩展：

1. 第二种天气与更长的水源规划。
2. 食物腐坏和烟熏/晾晒。

4. 临时远征营地和更大的导航区域。
